# MY_PORTFOLIO

A modern, animated AI/ML portfolio built with **React + Vite**.

## 🚀 Features
- Cinematic splash screen intro
- Custom animated cursor
- Matrix rain canvas background
- Typewriter effect hero
- Scroll-triggered animations
- Animated count-up stats
- Active nav link highlighting
- Project filter (AI / Web / Java)
- Resume modal viewer + download
- 3D holographic about photo
- Back-to-top rocket button
- Glowing scroll progress bar
- Fully responsive design

## 🛠️ Tech Stack
React · Vite · CSS · Framer-style animations
